Roadmap

這裡會列出我在準備面試和學習的過程中的準備資料

Database​

  • CMU Database Group : 經典的資料庫開放式課程 (15-445/645、15-721) 以及一些資料庫相關的研究
  • The Internals of PostgreSQL : 深入了解 PostgreSQL 的內部運作

Design Pattern​

  • Refactoring Guru : 提供各種設計模式的介紹與範例

Interview​

對於常見的 SWE 面試來說，通常會有幾種形式，分別是履歷關、行為面試、資料結構與演算法、物件導向設計、系統設計以及可能的針對職位的關卡等，以下是我覺得不錯的資源

Resume Preparation​

  • LaTex Resume : 我使用的 Latex 履歷模板
  • resumes.fyi : 提供各種履歷範本的網站

Data Structure & Algorithm​

  • Hello 演算法 : 新手友善的演算法入門指南
  • labuladong 的算法笔记 : 同樣是演算法筆記，但蠻多內容需要收費
  • NeetCode : 經典題單
  • Leetcode 101 : 同樣是題單，但也有解題思路

Object Oriented Design​

  • grokking-the-object-oriented-design-interview : 經典 OOD 課程的紙本

System Design​

  • Hello Interview : 系統設計的教學網站，也有 youtube 頻道，我覺得圖文都還蠻不錯的
  • Jordan Has No Life : youtube，除了有常見的系統設計問題外，還有一些實際的組件分析和論文閱讀

Behavioral Questions​

  • 常見的行為面試問題蒐集 : HackMD 上前人列出的常見問題，有給一些範例解答

Other people's Interview Resources​

  • Leetcode刷題學習筆記-心得統整

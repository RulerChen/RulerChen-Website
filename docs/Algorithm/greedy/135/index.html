[LC] 135.Candy (Hard)

題目​

有 n 的小孩站成一排，每個小孩都有一個評分。你需要按照以下要求，給這些小孩分糖果：

  1. 每個小孩至少一顆糖果
  2. 如果一個小孩的評分比他旁邊的小孩高，那麼他的糖果數也要比旁邊的小孩多

依據以上要求，請問你至少需要多少顆糖果？

Input: [1,0,2]
Output: 5
Explanation: [2,1,2]

解法​

由於每個孩子最少可以得到一顆糖果，所以我們可以先給每個孩子一顆糖果。 接著我們從左到右遍歷一次，如果右邊的孩子評分比左邊的孩子高， 那麼右邊的孩子的糖果數就是左邊的孩子的糖果數加一， 這樣可以保證右邊的孩子的糖果數比左邊的孩子多。 再來我們從右到左遍歷一次，如果左邊的孩子評分比右邊的孩子高，且左邊的孩子的糖果數比右邊的孩子少， 那麼左邊的孩子的糖果數就是右邊的孩子的糖果數加一， 這樣就可以同時滿足左向右和右向左的條件。

class Solution {
public:
    int candy(vector<int>& ratings) {
        const int n = ratings.size();
        if (n == 1) return 1;

        vector<int> v(n, 1);

        for (int i = 1; i < n; ++i) {
            if (ratings[i] > ratings[i - 1]) {
                v[i] = v[i - 1] + 1;
            }
        }
        for (int i = n - 1; i > 0; --i) {
            if (ratings[i - 1] > ratings[i]) {
                v[i - 1] = max(v[i - 1], v[i] + 1);
            }
        }
        return accumulate(v.begin(), v.end(), 0);
    }
};

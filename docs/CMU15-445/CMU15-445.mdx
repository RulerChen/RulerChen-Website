這個系列是我對 CMU 15-445/645 Introduction to Database Systems 的筆記，版本是 2024 Fall。

這門課會從 DB 的底層開始做一些大致的介紹，包括存儲層、索引、查詢處理、交易處理、跟簡單的分布式系統等等，對於應用的場景比較少著墨，作業會用 C++ 實作一些重要的 DB 組件，如 B+ Tree 等等。
對於想從底層了解 DB 的實作的人來說，這門課是非常好的選擇。

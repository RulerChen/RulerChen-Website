這裡會依據主題列出我在學習過程中看過的公開課程、書籍、Blog 等等的資料

## General

- [CS 自学指南](https://csdiy.wiki/) : 中文的自學指南，列出了許多經典的國外開放式課程以及資源

## Database

- [CMU Database Group](https://db.cs.cmu.edu/) : 經典的資料庫開放式課程 (15-445/645、15-721) 以及一些資料庫相關的研究，可以參考我的[筆記](/docs/CMU15-445/)

## Interview

### Preparation

- [English for Career Development](https://www.coursera.org/learn/careerdevelopment) : Coursera 上的課程，簡單介紹面試流程以及準備的方向，跳著看花 1-2 小時看完即可
- [LaTex Resume](https://www.overleaf.com/latex/templates/software-engineer-resume/gqxmqsvsbdjf) : Overleaf 上的履歷模板，可以用來製作自己的履歷

### Data Structure & Algorithm

- [Hello 演算法](https://www.hello-algo.com/zh-hant/chapter_hello_algo/) : leetcode 入門指南，有繁體中文版，可以快速了解基本的演算法
- [labuladong 的算法笔记](https://labuladong.gitbook.io/algo/) : 同樣是演算法筆記，但沒有繁中版

### System Design

- [Hello Interview](https://www.hellointerview.com/) : 系統設計的教學網站，但現在有些內容要收費了

### Behavioral Questions

- [常見的行為面試問題蒐集](https://hackmd.io/@6PPVul2mS7OX2GuGXlUCDA/behavior_questions) : HackMD 上前人列出的常見問題，有給一些範例解答

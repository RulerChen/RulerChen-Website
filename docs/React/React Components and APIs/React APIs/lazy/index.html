lazy

Syntax​

import { lazy } from "react";

const MarkdownPreview = lazy(() => import("./MarkdownPreview.js"));
  • React 會在第一次 load 的時候去 import component ，且回傳的是一個 promise 或具有 then 方法的物件。
  • load 完成後， React 會 cache 這個 component ，所以之後再次使用時，就不會再次 load。
  • lazy 應該被放在 module 的最上層。

Usage​

Lazy-loading components with Suspense​

import { lazy } from "react";

const MarkdownPreview = lazy(() => import("./MarkdownPreview.js"));
<Suspense fallback={<Loading />}>
  <h2>Preview</h2>
  <MarkdownPreview />
</Suspense>

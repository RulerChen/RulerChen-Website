併查集(Disjoint Set)

一、簡介​

Disjoint Set 本身就是一種 Set，表示了每組 Set 都是互斥的，比如 A={1, 2}、B={3, 4}， 則 A 和 B 即為 Disjoint Set，在最差的情況下會退化成鍊表，因此我們要採取以下幾種方法來避免。

  1. Path Compression : 將走過的節點都指向父節點，我們可以用遞迴來完成
  2. Union by rank : 將 rank 比較小的 set 加入 rank 比較大的 set 裡面

透過以上這兩種方法來降低 Disjoint Set 的 find 和 union 複雜度到 O(1)，而建構的複雜度則為 O(n)。

Disjoint Set.cpp
class DisjointSet {
   public:
    // 輸入大小
    DisjointSet(int n) {
        rank.resize(n, 0);
        parent.resize(n, 0);

        for (int i = 0; i < n; ++i) {
            parent[i] = i;
        }
    }
    // 輸入要找的數字， return parent 並做 path compression
    int find(int n) {
        if (n == parent[n]) {
            return n;
        }
        return parent[n] = find(parent[n]);
    }
    // union by rank，結合兩個 set
    // 如果成功 return true，如果本來就在一個 parent 下就 return false
    bool unionSet(int a, int b) {
        int parent_a = find(a);
        int parent_b = find(b);
        if (parent_a == parent_b) return false;

        if (rank[parent_a] > rank[parent_b]) {
            parent[parent_b] = parent_a;
        } else if (rank[parent_a] < rank[parent_b]) {
            parent[parent_a] = parent_b;
        } else {
            parent[parent_b] = parent_a;
            rank[parent_a]++;
        }
        return true;
    }

   private:
    vector<int> parent;
    vector<int> rank;
};

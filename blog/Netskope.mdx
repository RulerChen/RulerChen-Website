2025 Software Summer Intern @ Netskope

## 前情提要

書接上文，由於當時在旋轉的人很多都跳到了 Netskope，考量到想在當兵前多看看不同的方向並看看有沒有 return offer 可以拿，所以後來就投了一下想說試看看，也沒有特別準備，只報了一下旋轉的 mentor 的名字就上了。

這是 Netskope 在臺灣的第一屆實習生，據他們所說之後會想招募更多的實習生。實習的內容可能會根據我們的回饋做很多的改變，所以如果有幸有人看到這篇的話可以不用太參考實習內容。

## 面試過程

面試的過程可以分為 OA 跟 BQ 兩面。

OA 其實還蠻簡單的，大概就是 easy ~ medium 左右的程度，比較特別的是它還要開鏡頭寫。

BQ 會有兩個臺灣的 EM 用英文跟你面試，所以其實沒有甚麼語言障礙。
主要就是詢問一些履歷跟一些行為面試的問題，像是你在團隊中遇到衝突怎麼處理、你覺得你最大的優點是什麼之類的，以及一些少部份的技術問題，我記得是問跟 Restful API 相關的問題。

## 實習過程

這次臺灣的 intern 大概有 5 (-1) 個人，分成了兩個不同的 project，五個人都會各自分配到一個 manager，project 也有 project manager 負責整體的進度跟協調。

我們這組負責的是一個內部監控系統的系統，主要是用來監控公司內部各種服務的健康狀態，這是一個完整的前後端專案，可以學習前後端以及使用 k8s + helm chart。

我們會有固定的跟 project mentor 的 bi-weekly meeting，也會每周定期跟各自的 manager 一起 1 on 1，討論進度跟遇到的問題。

另一組則是負責做一個 github action 的 workflow，透過接內部的 API 來讓大模型 review 單元測試寫的好不好，那邊做的東西感覺比較無聊，原本應該是一個可以很快結束的專案，但因為後來有一位實習生出了一點問題所以那邊就拖了很久。

結束之後每個人都要用英文作一個簡短的 presentation，介紹自己做了什麼東西跟有甚麼 takeaway。

## 實習心得

先講這裡的優點吧，公司有提供午餐，大部分的午餐都還蠻好吃的，同事也都非常友善，願意提供很多經驗跟職涯的建議，manager 跟 mentor 也都蠻不錯的，不太會有不被重視的感覺。

但除此之外，由於是第一屆的實習，整體內容沒有太符合我的預期，所以我覺得這次的實習體驗是有點可惜的。

首先是專案吧，我們做的專案跟公司的其他業務沒什麼關係，並且是實習生要自己合作，mentor 雖然會給予建議但是不會參與開發，所以其實會有一種自己在做 side project 的感覺，自己並沒有感覺到這兩個月有甚麼 technical growth。
除此之外，我覺得因為公司比較大，所以其實在很多事情上限制都比較多，有很多東西在最後做出來的時候都覺得不是很完整，比如說網址甚麼的都沒有，做的時候就覺得這東西就算我弄完了其實也不太會有人用，所以其實做的蠻沒有成就感的。

第二件事情比較有趣，就是有一位實習生因為對專案不滿意就直接擺爛的故事，我覺得這件事也讓我學會從不同的角度去看事情。

故事的背景是有一位實習生 A 跟實習生 B 是同一組，但是 A 一開始的時候就對實習內容就不是很滿意，後來就把怨氣發洩在跟他同組的 B 上面，從此之後他們就徹底決裂了。
而這位 A 也就從此開始帶自己電腦來公司然後自己做自己的東西，然後也不太跟我們其他人交流。

從整件事情的角度來看，整件事情肯定是 A 的問題最大，畢竟我們不可能遇到不喜歡做的事情就擺爛，我覺得這樣的態度是非常不成熟的。
從 B 的角度說，他除了要幫 A 擦屁股以外，還是忍受 A 坐在自己旁邊每天遲到早退還做自己的事情，這樣的情況下其實不管是誰都會心生不滿，甚至對公司有蠻多怨言的。

如果從我自己的角度來說，我一開始對這件事情的冷處理方式也蠻不滿的，直接跟 A 說叫他不要做了應該是快且最直接的處理方式。
但從管理的角度看，由於這是第一屆的實習，如果直接把 A 開除的話，可能會讓未來的實習生覺得這個公司不太友善，進而影響到公司的形象。
不過不管如何，我覺得 Netskope 的管理層可以考慮更積極的介入，像是早點讓 B 轉組，或是問 A 要不要去做其他的 project 等等 (雖然我覺得擺爛的人應該不太可能會想要去做其他的 project)，至少讓最後的結果不會是有人每天在公司做一些很誇張的事情。

最後一天跟我的 manager 溝通完之後，我發現站在不同的角度來看這件事情，會發現每個人都有自己的立場跟難處，這也讓我學會在未來遇到類似的事情時，可以更冷靜的去思考問題。

最後還有一件事，不知道是不是因為有白癡兵役的關係，雖然我的 manager 在最後的評價中給了我一個蠻正面的評價，但還是沒有 return offer，感覺有點可惜。

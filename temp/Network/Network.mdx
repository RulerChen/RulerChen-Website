這系列主要是參考馬薩諸塞大學的課程，教授是計算機網路領域經典教材 (Computer Networking - A Top-Down Approach) 的作者。

本課程的實作部分較少，只有基於 Wireshark 的一些 lab，內容上會逐層介紹網路的各個層級，包括應用層、傳輸層、網路層和鏈路層等。
